# RoPhim Blocker - Extensions Chrome, Edge, Brave ...

**Tính năng chính:**

* **Chặn banner & ảnh GIF của RoPhim:** Vô hiệu hóa các loại banner, ảnh GIF nhấp nháy từ máy chủ quảng cáo của RoPhim và các nguồn bên thứ ba
* **Chặn nhảy trang (Popup):** Ngăn chặn việc tự động mở tab quảng cáo khi tương tác
* **Chặn chuyển hướng:** Vô hiệu hóa lệnh tự động chuyển trang từ RoPhim đến các domain quảng cáo
* **Dọn dẹp giao diện:** Ẩn banner, bảng thông báo, quảng cáo che màn hình ...
* **Bộ lọc:** Sử dụng quy tắc chặn quảng cáo lấy từ **ABPVN List**

**Cài đặt:**

1. Tải mã nguồn về máy: [RoPhim_Blocker.zip](https://github.com/bibicadotnet/RoPhim_Blocker/releases/latest/download/RoPhim_Blocker.zip)
2. Truy cập `chrome://extensions/` trên trình duyệt.
3. Bật **Chế độ dành cho nhà phát triển** (Developer mode).
4. Chọn **Tải tiện ích đã giải nén** (Load unpacked) và trỏ vào thư mục chứa code.
